
# LeetCode & Coding Challenge Solutions

This repository is a collection of my solutions to various coding problems from platforms like LeetCode, NeetCode, and others. The solutions are implemented in Go and JavaScript, and the repository is organized to reflect the problem sets and categories.

## ðŸš€ Getting Started

To run the solutions in this repository, you will need to have Go and/or Node.js installed on your machine.

### Go

To run a Go solution, navigate to the file's directory and use the `go run` command:

```bash
go run <filename>.go
```

### JavaScript

To run a JavaScript solution, you will need Node.js. Navigate to the file's directory and use the `node` command:

```bash
node <filename>.js
```
